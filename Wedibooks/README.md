# Features(Backend)
- User
회원가입 / 로그인 기능 구현 - bcrypt 암호화, 회원가입 유효성 검사 구현(Post, Get)
- Product
Product 별 상세페이지 구현(Get)
Category 별 product list 구현(Get)
Main page 내 추천도서, 베스트 셀러 및 신간도서 list 구현(Get)
- Review
Review posting(Post)
Review 검색(Get)
- Cart
상품 장바구니 담기(Post)
장바구니 조회(Get)
- CSV file upload
